Help and feedback

Support

Get help with Imposter, report a bug, or share feedback about game flow, languages, categories, or generated words.

Report a bug

Include your device model, operating system version, app version, what happened, and the steps that caused the issue.

Generated word issue

Include the selected language, category, difficulty, and the word or clue that felt wrong, unsafe, unclear, or too obscure.

Feature feedback

Share what would make the pass-and-play flow faster, clearer, or more fun for your group.

Contact options

Use the contact method listed in the app store listing, or open a GitHub issue at github.com/TolibovZafarali/imposter/issues.

Before you report a problem

  • Make sure you are using the latest version of the app.
  • Restart the app and try the same flow again.
  • Check whether the issue happens only with one language or category.
  • Avoid sending private player names or other personal information.
